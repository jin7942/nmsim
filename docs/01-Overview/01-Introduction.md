# NMSIM - NMS 개발용 SNMP 시뮬레이터

## 개요

NMSIM은 NMS(Network Management System) 개발 시 실제 네트워크 장비 없이 SNMP 연동을 테스트할 수 있는 시뮬레이터입니다.

## 목적

- 실제 장비 없이 NMS SNMP 연동 테스트
- 다양한 네트워크 장비 시뮬레이션
- 장애 상황 재현 및 테스트
- 개발 생산성 향상

## 타겟 사용자

- NMS 개발자
- 네트워크 모니터링 솔루션 개발자
- SNMP 연동 테스트가 필요한 개발팀

## 핵심 가치

| 가치 | 설명 |
|------|------|
| 실제 장비 대체 | 실제 장비와 동일한 SNMP 응답 |
| 표준 MIB 지원 | RFC1213-MIB, IF-MIB 등 표준 MIB 완벽 지원 |
| 간편한 설정 | GUI로 장비 추가/설정 |
| 장애 시뮬레이션 | 타임아웃, 에러, 지연 등 다양한 상황 재현 |
