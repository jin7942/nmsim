# 빠른 시작 가이드

## 1. VM 시작

VM 부팅 후 자동으로 NMSIM 서비스가 시작됩니다.

## 2. 웹 UI 접속

브라우저에서 `http://{VM_IP}:8080` 접속

## 3. 장비 추가

1. 좌측 메뉴에서 **장비** 클릭
2. **+ 장비 추가** 버튼 클릭
3. 정보 입력:
   - IP 주소: `10.0.0.100`
   - 장비명: `Test-Router`
   - 타입: `Router`
4. **저장** 클릭

→ 즉시 해당 IP로 SNMP 응답 가능

## 4. SNMP 테스트

### 방법 A: 웹 UI 테스트 콘솔

1. 좌측 메뉴에서 **테스트** 클릭
2. 대상 IP 선택: `10.0.0.100`
3. OID 입력: `1.3.6.1.2.1.1.1.0`
4. **실행** 클릭

### 방법 B: 커맨드라인

```bash
# NMS 서버 또는 다른 PC에서
snmpget -v2c -c public 10.0.0.100 sysDescr.0
snmpwalk -v2c -c public 10.0.0.100 ifTable
```

## 5. OID 값 수정

1. 장비 목록에서 장비 선택
2. **OID 편집** 탭 클릭
3. 수정할 OID 선택
4. 값 변경 후 **적용**

## 6. 동적 값 설정

OID 편집에서 **동적 변화** 활성화:

- 카운터: 지속적으로 증가
- 게이지: 범위 내 변동
- 업타임: 시간 증가

## 예시: 트래픽 시뮬레이션

```
OID: ifInOctets.1
동적 규칙:
  - 타입: counter
  - 증가량: 10000
  - 간격: 1초
```
