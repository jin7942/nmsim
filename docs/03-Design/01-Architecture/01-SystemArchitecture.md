## 계층 설명

### 1. Presentation Layer (Web UI)
- React 기반 SPA
- 장비 관리, OID 편집, 테스트 콘솔, 로그 뷰어

### 2. Application Layer (FastAPI Backend)
- REST API 제공
- WebSocket 실시간 통신
- 비즈니스 로직 처리

### 3. Domain Layer
- Device Manager: 장비 CRUD
- MIB Manager: MIB 정의 관리
- Request Logger: 요청 로깅

### 4. Data Layer
- SQLite 데이터베이스
- 장비 설정, OID 값, 요청 로그 저장

### 5. Infrastructure Layer (SNMP Engine)
- asyncio 기반 비동기 처리
- 가상 에이전트 풀
- 동적 값 계산 엔진
