# 기능 요구사항

## FR-001: SNMP 프로토콜 지원

### 설명
표준 SNMP 프로토콜을 지원하여 NMS와 연동할 수 있어야 한다.

### 세부 요구사항
| ID | 요구사항 | 우선순위 |
|----|---------|---------|
| FR-001-01 | SNMPv1 GetRequest 지원 | 필수 |
| FR-001-02 | SNMPv2c GetRequest/GetNextRequest/GetBulkRequest 지원 | 필수 |
| FR-001-03 | SNMPv3 인증(MD5/SHA) 지원 | 선택 |
| FR-001-04 | SNMPv3 암호화(DES/AES) 지원 | 선택 |
| FR-001-05 | SetRequest 지원 | 선택 |

---

## FR-002: 표준 MIB 지원

### 설명
표준 SNMP MIB의 OID에 대해 유효한 응답을 반환해야 한다.

### 세부 요구사항
| ID | 요구사항 | 우선순위 |
|----|---------|---------|
| FR-002-01 | RFC1213-MIB (system, interfaces) 지원 | 필수 |
| FR-002-02 | IF-MIB (ifXTable) 지원 | 필수 |
| FR-002-03 | HOST-RESOURCES-MIB 지원 | 권장 |
| FR-002-04 | ENTITY-MIB 지원 | 권장 |
| FR-002-05 | LLDP-MIB 지원 | 권장 |

---
